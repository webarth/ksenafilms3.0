# 🎬 Studio N - Landing Page Profissional

Landing Page moderna e responsiva desenvolvida para apresentar serviços de **Filmmaker e Design Gráfico**, com foco em conversão, autoridade e apresentação profissional de serviços com valores.

---

## 🚀 Sobre o Projeto

Este site foi desenvolvido utilizando:

- HTML5
- TailwindCSS
- JavaScript
- Lucide Icons
- Google Fonts

A proposta é apresentar de forma clara:

- Serviços disponíveis
- Valores
- Posicionamento profissional
- Contato direto via WhatsApp

---

## 🎨 Identidade Visual

O projeto utiliza uma paleta moderna baseada em:

- Verde escuro
- Verde esmeralda
- Preto
- Cinza escuro

Visual minimalista, premium e focado em contraste.
